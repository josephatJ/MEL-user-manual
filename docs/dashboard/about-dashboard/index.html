About MEL Dashboard

MEL Dashboard is a one stop center for viewing well organised or grouped visualizations (Charts & Tables) from different KPIs. Its a one stop center since one can do some more analysis via changing Data Dimensions to meet his/her needs. Can also download the data into different formats including graphical view, excel and pdf.
