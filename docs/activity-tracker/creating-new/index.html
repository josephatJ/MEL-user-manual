Designing Activity Tracker

Cloning existing#

This is a process of making a copy of existing design, then proceed with some editings

  1. Open Clone

  2. Define year to take the copy of selected activity tracker year

  3. Save

Steps for New Year of Activity Tracker#

The following are the steps for creating Activity Tracker, but if you want to make a COPY of previous years configurations, you can use a clone feature

1. Adding new year#

1.1. Login into MEL System

1.2. Open Reports module

1.3. Go to Activity Tracker



1.4. Click + button to add new year for activity tracking

1.5. Select year of action



1.6. Save, then close. A new year will be added.

2. Add Objective#

The following are the steps for adding new Objective

2.1. Click Add Objective feature

2.2. Feed Objective details as indicated in the screenshot

2.2. Save the Objective

3. Add Outcome#

The following are the steps for adding new Outcome

3.1. Click on more Options to see avaialable actions, then click Add Outcome

3.2. Feed Outcome details as indicated in the screenshot. Details include name, label, description, indicator. Also one will be required to enter the baseline and target for selected indicator(s).


3.3. Save the Outcome

4. Add Output#

The following are the steps for adding new Output

4.1. Click on more Options to see avaialable actions, then click Add Output

5. Add Activity#

The following are the steps for adding new Activity

5.1. Click on more Options to see avaialable actions, then click Add Activity
